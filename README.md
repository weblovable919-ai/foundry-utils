# foundry-utils

Utility toolkit for Foundry EVM development

## Overview

A collection of utility functions for Foundry-based EVM development. Helper functions for testing, deployment, and common operations.

## Features

- Test helpers for Foundry
- Deployment utilities
- ABI encoding helpers
- Gas optimization tools
- Solidity cheatcode wrappers

## Installation

```bash
pip install foundry-utils
```

## Quick Start

```python
import import foundry-utils
```
